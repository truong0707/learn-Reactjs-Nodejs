CSS Position: Relative
(nghĩ đơn giản là giao diện đè)
(ví dụ khi ta vào trang facebook, khi bấm vào thanh chức năng thông báo thì nó sẽ hiện 1 thanh nhỏ ra màn hình đè lên page)
(tóm lại nó có công dụng thiết lập các vị trí hiển thị element lên wedsite)

Có các thuộc tính của Position như sau:

  • Realative(vị trí tương đối)
  • Absolute (vị trí tuyệt đối)
  • Fixed (vị trí phụ thuộc vào khung trình duyệt)
  • Fixed (vị trí bám dính, nó sẽ phụ thuộc với khung trình duyện , và thuộc tính này ko đc khuyến cáo sử dụng)
